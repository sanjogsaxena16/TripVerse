# TripVerse 🌍✈️

**TripVerse** is a modern, responsive web application built to simplify travel planning across India. The platform integrates travel booking, trip planning, and unified transport options into one seamless, premium user experience.

## ✨ Features

- **Multi-modal Transport Booking:** Search and book flights, trains, buses, and cabs all in one place.
- **Trip Planner:** Organize your upcoming vacations by defining dynamic trip itineraries and destinations.
- **Premium User Interface:** Experience a modern UI featuring smooth micro-animations, rich gradient designs, and sleek glassmorphic elements.
- **Fully Responsive Navigation:** Ensures seamless usability across all devices (desktop and mobile-friendly).
- **Authentication Pages:** Includes beautiful login and registration screens for secure user access.

## 🚀 Getting Started

Getting started with TripVerse is extremely easy since it is built with pure HTML, CSS, and vanilla JS. No build steps (like Webpack or Vite) are required.

To view the application:
1. Clone the repository or extract the project folder.
2. Simply double-click on `index.html` or open it with any modern web browser (Chrome, Edge, Firefox, Safari).
3. Navigate through the site to explore all the features seamlessly.

## 🛠️ Technology Stack

- **HTML5:** Semantic architecture for accessible and organized web pages.
- **Vanilla CSS3:** Highly curated gradients, 'Poppins', 'Inter' geometry, and modern hover states.
- **Vanilla JS:** Front-end orchestration for smooth anchor scrolling, element mockups, and dynamic navbar handling.
